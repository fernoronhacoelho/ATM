Projeto 4 ATM

Aluna: Fernanda Noronha Coelho dos Santos Marques - 190027762

O projeto consiste na aplicação de conhecimentos de programação orientada a objetos, Python e JSON para o desenvolvimento de um sistema bancário que possui requisitos que serão destacados abaixo. Aqui foi aplicado um projeto de arquitetura mcv (Models, Controllers e Views), onde temos os modelos que foram utilizados, a visualização do projeto em terminal e o controle do banco de dados em um arquivo separado. 

Observação importante: 
* Para acessar um gerente que é responsável por registrar o cliente e atualizar os dados cadastrais, use o login: 0000 e senha: gerente. 
* Ao finalizar as ações do gerente basta sair do sistema e rodar o programa de novo logando no login e senha do cliente desejado.
* Sempre que você sair do sistema será necessário rodar o programa novamente.
